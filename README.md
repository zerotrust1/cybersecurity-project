# CyberProject

A collection of cybersecurity-related tools and projects.

## Projects

### [Cipher Identifier](./chiper-identifier)
A real-time web application to detect and decode common encodings and classical ciphers.

## Features (Planned & Current)
- [x] Base64, Base32, Hex, Binary detection
- [x] Morse Code decoding
- [x] Classical Ciphers: Caesar, Atbash, Affine, Bacon, Vigenère
- [ ] More advanced cryptanalysis tools
- [ ] Steganography detection
